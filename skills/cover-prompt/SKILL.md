# cover-prompt

## 目的

为公众号文章、短视频封面生成图片提示词。

目标不是做炫酷 AI 海报，而是做能提高点击率、符合文章意思的封面。

## 默认比例

公众号封面：

```text
2.35:1
```

短视频封面：

```text
3:4
```

文章配图：

```text
16:9
```

## 封面原则

1. 人物要真实，不要塑料 AI 感。
2. 场景要合理，饭盒、手机、快递箱不能乱飘。
3. 标题必须和文章核心意思一致。
4. 不要为了视觉冲击，偏离文章内容。
5. 快递员形象要生活化，不要变成科幻英雄。
6. 封面要一眼看懂冲突。

## 常用场景

```text
快递站门口
出租屋小桌
楼道
小区门口
电动三轮车旁边
中午吃盒饭
夜晚路灯下看手机
```

## 提示词结构

```text
主体人物：一个普通中国快递员……
场景：……
动作：……
情绪：……
画面冲突：……
风格：真实纪实摄影风格，生活化，电影感光影，高细节
限制：无文字，无水印，比例……
```

## 标题处理

如果用户要求图片里加标题，要注意：

```text
标题短
字要大
不要贴边
不要遮挡脸
不要和文章意思冲突
```

## 禁止

```text
不要让物品浮空
不要把用户照片变成陌生 AI 脸
不要出现香烟，除非用户明确要求
不要过度科幻
不要堆满复杂 AI 元素
```
