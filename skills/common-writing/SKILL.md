# common-writing

## 目的

用于公众号文章、短文、推文的基础写作。

目标不是写得高级，而是写得像真人，有经历，有判断，有刀口。

## 适用场景

```text
公众号长文
热点评论
AI 工具使用心得
普通人赚钱思考
戒烟、熬夜、自控类内容
```

## 写作原则

1. 先说人话，再说观点。
2. 先讲真实经历，再讲道理。
3. 不要一上来铺大背景。
4. 不要假装自己什么都懂。
5. 复杂概念必须用生活类比翻译。
6. 结尾必须落到一个清楚判断。

## 结构建议

```text
开头：直接抛出问题或反常识判断
中段：用自己的场景切入
展开：拆利益关系、机制、误区
回扣：普通人能怎么办
结尾：一句能当标题的金句
```

## 语言风格

```text
我送快递的
说实话
这事最别扭的地方在于
你以为是在……其实是在……
普通人真正怕的不是……而是……
```

## 禁止写法

```text
在当今时代
随着科技的发展
赋能
闭环
底层逻辑乱飞
首先其次最后机械排队
每句话一个段落
```

## 输出要求

默认保留用户原始表达的刀口。

修改时优先做三件事：

```text
删废话
补逻辑
压 AI 味
```

不要把用户原文洗成标准作文。
