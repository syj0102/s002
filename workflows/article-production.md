# article-production

## 公众号文章生产流程

这个 workflow 用于把一个选题变成一篇公众号文章。

## 1. 选题判断

读取：

```text
skills/topic-selection/SKILL.md
```

先判断：

```text
这个选题值不值得写？
它回答的好问题是什么？
和普通人有什么关系？
用户有没有真实切口？
是否需要查证？
```

## 2. 素材整理

整理用户提供的碎碎念、录音转文字、新闻链接、截图内容。

不要急着重写。

先做：

```text
保留原话里的刀口
找出最有力的场景
找出最像人的句子
删除重复废话
```

## 3. 文章成稿

读取：

```text
skills/common-writing/SKILL.md
contexts/reverse-life-hub.md
```

按快递员第一人称写。

## 4. 事实核查

如果涉及新闻、产品、日期、公司、价格、政策，必须查证。

不能凭标题猜「最近」。

## 5. 封面提示词

读取：

```text
skills/cover-prompt/SKILL.md
```

根据文章核心冲突生成封面图提示词。

## 6. 最终检查

检查四件事：

```text
有没有 AI 味
有没有编造场景
标题和文章是否一致
结尾金句是否能当标题
```
